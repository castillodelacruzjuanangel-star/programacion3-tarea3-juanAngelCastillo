# programacion3-tarea3-juanAngelCastillo
tarea sobre excepciones, try catch y excepciones personalizadas.

El programa tiene como funcionalidad validar las entradas de usuario(nombre, edad, correo electronico y sueldo)
existe un catch para cada validacion asi como excepciones personalizadas para cada entrada.

El programa cuenta con un menu simple que contiene: registrar y salir
Al seleccionar registrar se tendra que rellenar los siguientes campos
1- nombre
2- edad
3- correo electronico
4- sueldo

Se valida que:
-El nombre no este vacio o que tenga menos de 2 caracteres(sin contar espacios)
-La edad no puede ser menor de 18 años ni mayor 100 años
-El correo electronico debe tener '@' y '.'
-El sueldo no sea menor o igual a 0
